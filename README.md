# 💰 Home Banking - Proyecto en JavaScript

Este proyecto es una simulación simple de un sistema de Home Banking desarrollado con **HTML, CSS y JavaScript puro**, como parte de una práctica para aprender las bases de la programación web y la interacción con el usuario.

---

## 🧩 Funcionalidades

- Inicio de sesión con usuario y contraseña.
- Consulta de saldo.
- Depósito de dinero.
- Retiro de dinero.
- Registro y visualización de los últimos movimientos.
- Estilo básico responsivo con CSS.
- Interacción mediante `prompt`, `alert` y `confirm`.
- Uso de consola (`console.log`) para depuración.

---

## 📚 Tecnologías utilizadas

- **HTML5** para la estructura de la página.
- **CSS3** para el diseño visual.
- **JavaScript** (ES6+) para la lógica del sistema.

## 🚀 Cómo usar el proyecto

1. Cloná o descargá el repositorio.
2. Abrí `index.html` en tu navegador.
3. Hacé clic en "Iniciar sesión".
4. Ingresá:
   - **Usuario:** `cliente`
   - **Contraseña:** `1234`
5. Interactuá con el menú de operaciones.
